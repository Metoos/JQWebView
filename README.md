# JQWebView
1、对WKWebView进行一些封装<br>
2、添加网页加载进度条，可自定义颜色。<br>
3、支持侧边滑动返回上一级网页。<br>
最新更新删除UIWebview及有关部分的所有代码，完全使用WKWebView。<br>
cocoaPods安装：<br><br>
pod 'JQWebView', '~> 1.1.1'
<br><br>
搜索不到，可先执行如下命令行后重试：<br>
pod repo update<br>
