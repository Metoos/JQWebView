# JQWebView
1、iOS8.0及以上自动使用WKWebView，系统为iOS8.0以下时使用UIWebview<br>
2、添加网页加载进度条，可自定义颜色。<br>
3、支持侧边滑动返回上一级网页。<br>
cocoaPods安装：<br><br>
pod 'JQWebView', '~> 0.0.1'
<br><br>
搜索不到，可先执行如下命令行后重试：<br>
pod repo update<br>
